SumFields Function

sumFields function enables you to sum defined fields

Syntax:

Public Sub sumFields(ByVal ParamArray str_Line() As String)
It takes only one parameter
1- str_Line  is a  parameter array , so you can add as many summed fields  as you want , delimetered by comma ","

Example:

Dim oRep As New DynamicComponents.DynamicReport()

oRep.sumFields("Quantity", "Value", "DiscountValue", "NetValue")
